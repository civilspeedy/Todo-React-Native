# Todo React-Native

## Description

Task-tracking app developed for my final year unit "Ubiquitous Computing". The purpose of the application was to provide students with an easy and streamlined user experience for creating and tracking tasks, along with features designed to keep the user motivated, such as motivational messages and a points system.

## Results

For this assignment I was awarded a 1st.
